# Traffic Checklist

Run this checklist before recommending or publishing changes.

## Content and CTA Checks

- Keep one dominant value statement in the hero.
- Place both YouTube and Patreon CTAs above the fold.
- Add one mid-page CTA pair after trust-building content.
- Match CTA copy to benefit, not generic verbs.

## Metadata and Discovery Checks

- Ensure page has a unique `<title>`.
- Ensure meta description explains value + audience.
- Add OpenGraph title, description, and image tags.
- Keep heading hierarchy valid (one H1, meaningful H2 blocks).

## Trust and Conversion Checks

- Include a short proof block (results, testimonials, or examples).
- Embed relevant media where possible (YouTube teaser/video).
- Add an email capture option for non-converting visitors.
- Keep outbound links obvious and mobile-friendly.

## Tracking and Experiment Checks

- Add outbound event tracking for YouTube and Patreon links.
- Add campaign parameters when marketing source is known.
- Define one A/B copy test for hero + CTA wording.
- Record baseline metrics before shipping.
